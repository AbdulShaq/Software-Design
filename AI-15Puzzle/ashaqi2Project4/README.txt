This is a simple 15 puzzle game using javafx. The user is challenged to solve the puzzle.
If the user solves it they are prompted that they won and are given the option for a new game or to exit.
If the user cannot solve it they are given 2 options, either a new puzzle or to choose between to solving
methods which implement the A* solver algorithm. If the user uses the algorithm they are given the option
to get a hint of the next 10 steps or to solve the puzzle fully.
