This is a simple Client-Server game set up using Javafx and implementing Multi-Threading. The Server Program
prompts for a Port number contains all the game logic and controls, while also keeping track of the number of 
clients playing the game and the results of each hand played by each client printed to the screen for the Server.
The Client Program launches prompting the user for both a port number and IP address, while checking for both inputs
to be valid. the user is then taken to the game table where they are prompted to make valid betting amounts and choose
what to bet on. Each hand will result in the cards being displayed to the user and the inforamtion of each hand displayed.
